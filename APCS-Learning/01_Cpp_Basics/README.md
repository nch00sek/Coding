# 01_Cpp_Basics

## 這週學習目標
- 熟悉 C++ 基本語法（變數、型態、運算子）
- 學會 if/else 條件判斷、for/while 迴圈
- 了解 cin/cout 輸入輸出
- 初步認識 STL vector 與 string

---

## 學習步驟

### 1. C++ 基本語法
- 變數宣告與型態（int, double, char, string）
- 運算子（+ - * / %）
- 輸入輸出（cin, cout）

### 2. 條件判斷與迴圈
- if/else、switch
- for 迴圈、while 迴圈

### 3. STL vector 與 string
- vector<int> 的宣告與基本操作
- string 基本操作（長度、取字元、拼接）
